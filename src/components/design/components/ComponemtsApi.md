- `title`: '页面标题',
- `bgcolor`: '#ffffff'

- `type`: `whitespace`
- `height`: 30,
- `bgcolor`: '#ffffff'

- `type`: `line`
- `style`: solid, 边框样式
- `padding`: '0' 页面边距

#### 图片广告

- `type`: `image-ad`
- `padding`: 0 页面边距
- `spacing`: 0 图片间距
- `width`: 30(百分比，在横向滚动时使用)
- `template`: 1: 一行一个， 2: 横向滚动
- `items`: 同上
